FASE 1: MORFOLOGIA (HE)

  • → Architettura: Diffusa vs circonscritta
  • → Cellularità: Bassa/moderata/alta
  • → Atipia nucleare: Lieve/moderata/severa
  • → Mitosi: Assenti/rare/frequenti
  • → Necrosi: Assente/presente
  • → Grading provvisorio: G2/G3/G4
Remember: Il grading morfologico è PROVVISORIO. Può cambiare con CDKN2A/B.

FASE 2: IHC

  • → IDH1 R132H: 90% mutazioni IDH1
  • → ATRX: Loss = astrocitoma
  • → p53: Overexpression (TP53 mut)
  • → Ki67: Index proliferativo
  • → GFAP: Conferma origine gliale
  • → Olig2: Positivo in oligodendrogliomi
Pitfall: IDH1 R132H negativo non esclude IDH-mutato! 10% ha altre mutazioni. Sequenzia se <40 anni.

FASE 3: BIOLOGIA MOLECOLARE

  • → FISH 1p/19q: Obbligatoria per oligo
  • → TERT promoter: C228T/C250T
  • → CDKN2A/B: Homozyg del = G4
  • → EGFR amp: FISH/MLPA
  • → +7/-10: FISH o array-CGH
  • → Methylation: Casi ambigui
TEST OBBLIGATORI:
• 1p/19q FISH se ATRX preserved
• CDKN2A/B per grading
• IDH seq se R132H neg + <40 anni

📋 TIPS PRATICI

  • Ordina sempre IHC panel base (IDH1 R132H + ATRX + p53 + Ki67) prima di decidere i test molecolari
  • FISH 1p/19q è OBBLIGATORIA per oligodendroglioma (non puoi diagnosticare solo con IHC)
  • Se IDH1 R132H negativo + età <40 anni → sequenzia IDH1/2 completo (10% mutazioni non-R132H)
  • CDKN2A/B cambia il grade e quindi il trattamento → SEMPRE da testare in IDH-mutati
  • Se tutto negativo (IDH-wt NOS) → methylation profiling prima di refertare definitivo
  • Pattern ATRX loss + 1p/19q codel = SEMPRE artefatto tecnico → ripeti FISH
